HTTP Client

HTTP Client is used by scripting services to call external services via HTTP.

Version 4.x

  • Module: http/v4/client
  • Alias: http/client
  • Definition: https://github.com/eclipse/dirigible/issues/15
  • Source: /http/v4/client.js
  • Facade: HttpClientFacade
  • Status: stable

Basic Usage

var httpClient = require("http/v4/client");
var response = require("http/v4/response");

var httpResponse = httpClient.get("https://services.odata.org/V4/Northwind/Northwind.svc/");

response.println(httpResponse.statusMessage);
response.println(httpResponse.text);
response.flush();
response.close();
