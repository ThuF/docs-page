{{ page.title }}

The Messaging Producer is an object which can send text messages to a Queue or Topic destination in the built-in Message Broker. This version is backed by the full fledged messaging broker Apache ActiveMQ.

Version 4.x

  • Module: messaging/v4/producer
  • Alias: messaging/producer
  • Definition: https://github.com/eclipse/dirigible/issues/91
  • Source: /messaging/v4/producer.js
  • Facade: MessagingFacade
  • Status: stable

Basic Usage

var producer = require("messaging/v4/producer");
producer.queue("queue1").send("Text Message");

Definition

Functions

Function Description Returns
queue() Returns an object representing a Message Queue Queue
topic() Returns an object representing a Message Topic Topic
